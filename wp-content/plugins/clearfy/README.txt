Powerful SEO plugin

== Description ==

Мощный плагин для WordPress +25 пунктов по улучшению Вашего сайта

== Installation ==

1. Upload `clearfy` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
